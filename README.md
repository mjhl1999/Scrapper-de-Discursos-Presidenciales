# Scrapper de Discursos Presidenciales



## Requirements

| Technology      | Version              |
|-----------------|----------------------|
| Beautifulsoup 4 | 4.5.3                |
| Requests        | 2.13.0               |


### To install required dependencies

1. Clone the repository

2. Move to the project folder

3. Use

> $ pip install -r requirements.txt <br>
